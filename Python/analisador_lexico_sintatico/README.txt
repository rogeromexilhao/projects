analisador lexico e sintático desenvolvido em python para ler linhas de codigo lalg(em um arquivo txt) 
fazendo com que demonstre em cada linha lida os tokens que foram achados e armazenando em um vetor de tokens
e logo apos os tokens são descoberto a analisador sintático é ativado e assim temos a analise sintática dos tokens recebidos

o codigo tambem identifica erros ex:
falta ';' na linha 
falta 'var'
...

para isso é necessario passar no codigo o arquivo txt que deseja fazer a leitura,
caso venha a fazer alguma alteração no arquivo txt não esqueça de salvar o arquivo

dessa maneira o programa vai ler o arquivo e exibir os tokens achados e se a analise sintática e lexica foi completa

caso algum dos analisadores identifique um erro o programa é finalizado na linha onde o erro foi encontrado
